# Coding Guidelines

Naming conventions, file layout and style rules followed by the experiments in this lab.

## Experiment README structure

Every mini-project README must answer these 8 questions, in this order:

1. **Quel est l'objectif ?**
2. **Pourquoi cette technologie est-elle importante ?**
3. **Quel matériel est utilisé ?**
4. **Comment fonctionne le système ?**
5. **Comment reproduire l'expérience ?**
6. **Quels résultats obtient-on ?**
7. **Quelles difficultés ont été rencontrées ?**
8. **Quelles améliorations sont possibles ?**

A ready-to-copy skeleton is available in [experiment-template.md](experiment-template.md).
